# 作用域链
by 若愚@饥人谷

## 先看面试题

## 题目1
```
var a = 1
function fn1(){
  function fn2(){
    console.log(a)
  }
  function fn3(){
    var a = 4
    fn2()
  }
  var a = 2
  return fn3
}
var fn = fn1()
fn() //输出多少
```


##题目2

```
var a = 1
function fn1(){
  function fn3(){
    var a = 4
    fn2()
  }
  var a = 2
  return fn3
}
function fn2(){
  console.log(a)
}
var fn = fn1()
fn() //输出多少
```
## 解密
1. 函数在执行的过程中，先从自己内部找变量
2. 如果找不到，再从**创建当前函数所在的作用域**去找, 以此往上
